User is working on Ubuntu Linux and is interested in professional-grade vocal/voice synthesis tools. Their key constraint is **automation** — they want whatever they pick to be drivable from Claude Code (CLI, API, scripting, or file-based authoring).

In our conversation they evaluated options in this order:
1. Asked for the most capable/professional voice synthesizer plugin → was recommended Synthesizer V Studio Pro as the singing-voice leader.
2. Asked whether Synth V works on Ubuntu → yes for Studio 1, but Studio 2 dropped Linux.
3. Asked whether Synth V can be automated by Claude Code → only partially (scripting API, no CLI/render API).
4. Asked whether Windows would help → marginally, mainly via Reaper CLI + VST3 plugin.

They value: terse, accurate, sourced answers; willingness to do extensive web research when asked; honest acknowledgment of limitations.
